# Changelog

All notable changes to this project will be documented in this file.

The project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html) (semver).

## [0.1.0] - Initial release

- Expand diplomatic transcriptions to full form via Gemini API or local (Ollama/rules)
- TEI and PAGE XML support
- GUI with input/output sync, progress bar, model selection
- CLI: expand, train, test-gemini
- Input→TXT and Output→TXT export
- Click block to select parallel block in other panel
